# Shrnutí projektu

Tento skill shrne aktuální stav TypeScript projektu.

Pomocí MCP filesystem serveru přečti všechny `.ts` soubory ve složce `src/` a:

1. Vypiš seznam souborů
2. Pro každý soubor uveď počet řádků a stručný popis co dělá (1 věta)
3. Na konci uveď celkový počet souborů a řádků

Výstup naformátuj jako markdown tabulku.
